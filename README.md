# BaiduMapSearch
Android百度地图POI检索，行政区边界、公交、线路规划查询，地理编码介绍

POI是兴趣点的意思，地图上标注的商铺、饭店、学校、银行、医院、车站等等都是POI。

地理编码：坐标和地址之间的转换。

demo中api详细介绍请阅读我的[zhh_happig的简书博客](http://www.jianshu.com/u/d82bd37b1d29)
